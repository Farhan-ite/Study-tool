# Pomodoro Timer ⏱️

A simple Pomodoro Timer desktop application built using **Python** and **Tkinter**.  
This app helps improve focus and productivity by following the Pomodoro Technique: focused work sessions followed by short and long breaks.

## Features
- Work, short break, and long break cycles
- Automatic session switching
- Visual countdown timer
- Check marks to track completed work sessions
- Reset button to restart the timer anytime
- Clean and minimal GUI

## How It Works
- 1 work session = 25 minutes (default can be changed in code)
- After every work session, a short break starts
- After 4 work sessions, a long break starts
- Check marks appear after each completed work session

## Technologies Used
- Python
- Tkinter (GUI)
- Math module
- Event-driven programming with `after()`
